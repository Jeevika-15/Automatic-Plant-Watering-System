# Automatic-Plant-Watering-System
An automatic plant watering system built using an ESP32, soil moisture sensor, relay module, and water pump. The system measures soil moisture in real time and activates the pump when the soil is dry, turning it off once adequate moisture is reached.

*Features*
1. Real-time moisture monitoring
2. Automatic pump control
3. Saves water and prevents over-watering
4. Simple, reliable hardware setup

*Components*
1. ESP32
2. Soil Moisture Sensor
3. Relay Module
4. Mini Water Pump
5. Jumper wires & power supply
